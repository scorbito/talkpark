# 디자인 브리프 — 오늘은 승요

> Google Stitch / Claude Design 등 AI 디자인 도구에 붙여넣어 사용하는 디자인 요청서.

---

## 1. 프로젝트 한 줄 요약

KBO 야구 직관(현장 관람) 기록·승률 통계·커뮤니티 웹 서비스. 모바일 우선(Mobile-first) 반응형 웹. 2025 KBO 관중 1,231만 명 시장 대상.

## 2. 타깃 유저

- 20~40대 KBO 야구팬, 시즌권/경기 직관 경험 있음
- 자기 직관 승률을 친구·SNS에 자랑하고 싶어 함
- 경기 일정·팀 순위를 자주 확인

## 3. 디자인 톤 & 무드

| 항목 | 방향 |
|---|---|
| 분위기 | 밝고 경쾌, 스포츠 에너지, 데이터를 자랑스럽게 |
| 컬러 사용 | 팀 컬러 강하게 활용, 단 가독성 해치지 않게 |
| 폰트 무드 | 한글 서체 우선, 숫자·통계는 굵고 임팩트 있게 |
| 레이아웃 | 카드 기반, 여백 넉넉, 한 화면 한 메시지 |
| 톤 레퍼런스 | 토스(통계 임팩트) + 인스타(피드 시각성) + 당근(친근함) |
| 피해야 할 톤 | 차갑고 기업적인 대시보드, 어두운 호러풍, 과한 일러스트 |

## 5. 타이포

- 헤드라인: Pretendard Bold / 24~40px (승률 숫자는 더 크게, 48~64px)
- 본문: Pretendard Regular / 14~16px
- 캡션: Pretendard Medium / 12px
- 숫자 강조 (승률·게임차): tabular-nums, 굵게

## 6. 글로벌 레이아웃

- 모바일 폭 360~414px 우선 / 태블릿 768px / 데스크톱 1024px+
- 하단 고정 4탭 네비게이션 (모바일)
- 데스크톱은 좌측 사이드바 4탭
- 페이지 좌우 패딩 16px, 카드 간 간격 12px

## 7. 4탭 네비게이션

| 탭 | 아이콘 | 라벨 | 역할 |
|---|---|---|---|
| 1 | 🏠 | 홈 | 내 직관 메인 (압도적 메인) |
| 2 | 📅 | 일정 | 월간/주간 경기 일정 (+ 팀순위 페이지로 진입) |
| 3 | 💬 | 커뮤니티 | 직관 후기 피드 |
| 4 | 👤 | 마이 | 내 직관 통계·후기·친구·설정 통합 진입 |

> 유저 랭킹은 Phase 2 도입 시 마이 탭 상단에 진입 카드 추가 또는 5탭 확장.
> 팀 순위는 별도 페이지로 존재 (탭 아님). 진입: 홈 "상세보기" / 일정 탭 우상단 / 경기 상세에서.

## 9. 핵심 화면 상세 스펙

### 9.1 홈 (로그인, 직관 있음) — 가장 중요

세로 스크롤 누적 구조.

```
┌─────────────────────────────┐
│ [로고]      🔔   👤        │  ← 헤더 56px
├─────────────────────────────┤
│                             │
│       ⚾  내 직관 승률      │  ← Hero Card
│                             │     팀컬러 그라디언트 배경
│         .617                │     승률 숫자 64px Bold
│      37승 23패 2무           │     서브 통계 14px
│                             │
│   [공유하기]   [등록하기]   │  ← 1차 CTA 2개
├─────────────────────────────┤
│ 오늘 두산 vs LG  19:30 잠실 │  ← 내 팀 경기 배너 (조건부)
├─────────────────────────────┤
│ 이번 주               전체→ │
│ ┌──┬──┬──┬──┬──┬──┬──┐      │  ← 미니 캘린더 가로 스크롤
│ │월│화│수│목│금│토│일│      │
│ └──┴──┴──┴──┴──┴──┴──┘      │
├─────────────────────────────┤
│ 팀 순위              상세→  │
│ 1  🦁 삼성  45-2-30  .600   │  ← 10팀 한 줄씩 컴팩트
│ 2  🐯 KIA   44-1-31  .587   │     내 메인팀 행만 컬러 하이라이트
│ ... (10팀 전체)             │
├─────────────────────────────┤
│ [홈] [일정] [커뮤니티] [마이] │  ← 하단 탭
└─────────────────────────────┘
```

**핵심 강조점:**
- Hero 승률 카드는 화면의 50% 차지, 팀 컬러 풀로 적용
- 승률 숫자는 무조건 가장 큰 시각 요소
- "공유하기" 버튼이 "등록하기"와 동등하게 메인 (바이럴 핵심)

**셀 배지 규칙:**
| 표시 | 의미 |

| 우상단 ⚾ | 내가 직관함 |
| 우상단 ⭐ | 직관 + 인증 (골드) |
| V/L 작은 라벨 | 직관 결과 (승=초록/패=빨강) |

## 11. 핵심 유저 플로우 3개

### Flow A: 신규 유저 첫 직관 등록 (가장 중요)
```
랜딩 → 카카오 로그인 → 닉네임 입력 → 메인팀 선택
   → 홈(빈 상태) → 직관 등록 모달
   → 등록 완료 → 공유 카드 자동 노출 → 카카오 공유
```

### Flow B: 티켓 인증 직관 등록
```
홈 → FAB 직관 등록 → 날짜 선택 → 경기 선택
   → 티켓 사진 업로드 → Vision 파싱 자동 → 인증 배지
   → 등록 완료 → 후기 작성 유도 → 공유 카드
```

### Flow C: 후기 작성 + 피드 공유
```
마이 → 내 직관 리스트 → 직관 카드 → 후기 작성
   → 사진 1~3장 + 본문 + 공개범위 → 등록
   → 커뮤니티 피드에 노출 → 친구 좋아요
```
