# 오늘은 승요

KBO 직관 기록, 승률 통계, 커뮤니티를 위한 모바일 우선 Next.js 웹앱입니다.

## 실행

```powershell
npm.cmd install
npm.cmd run dev
```

개발 서버 실행 후 `http://localhost:3000`에서 홈 화면을 확인합니다.

## 기준 자료

- 디자인 브리프: `data/design_brief.md`
- 디자인 시안: `data/ChatGPT Image 2026년 5월 5일 오후 06_14_00 (*.png)`
- 앱 어셋: `public/assets`
